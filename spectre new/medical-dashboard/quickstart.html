Professional Healthcare Monitoring System - Quick Start Guide

🚀 Getting Started

Get the dashboard up and running in minutes.

  1. Open index.html in your browser
  2. Enter any username and password
  3. Click "Sign In" to access the dashboard
  4. Explore the monitoring interface

🧪 Local Testing

For a better development experience, use a local server:

# Python 3 python -m http.server 8000 # Python 2 python -m SimpleHTTPServer 8000

Then visit: http://localhost:8000
