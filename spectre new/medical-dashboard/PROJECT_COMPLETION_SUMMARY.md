## 🚀 HOW TO USE RIGHT NOW

### Step 1: Open in Browser
```
Open: medical-dashboard/index.html
```

### Step 2: Login
```
Username: any text
Password: any text
(You'll be redirected to dashboard)
```

### Step 3: View Dashboard
```
Default Patient: John Anderson
Vitals: Auto-update every 5 seconds
Charts: ECG and Respiratory animate in real-time
```

### Step 4: Switch Patients
```
Click "Patients" in sidebar → Patient list loads
Click filter button (Critical/Warning/Stable)
Type in search box to filter by name/ID/diagnosis
Click "View Dashboard" on any patient
Dashboard reloads with that patient's data
```

### Step 5: Monitor Patient
```
Watch real-time vital updates
See patient-specific ECG patterns
Check activity log and alerts
Switch to another patient anytime
```
